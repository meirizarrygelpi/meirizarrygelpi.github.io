CHSH Game and Bell's Theorem 1

  • Tue 06 February 2018
  • Physics
  • #8.370.1x

I am taking a MOOC on quantum information from MIT (8.370.1x). The discussion on Bell's theorem involves a classical game with two players, Alice and Bob. I wrote a little bit of Python code to understand the results better.

The game consist of a referee that produces two randomly chosen classical bits \(a\) and \(b\) and hands Alice and Bob one each. Alice and Bob are separated. Alice produces another bit, \(x\), and Bob produces another bit \(y\). The game is won if the following condition is met:

$$ \operatorname{XOR}(x, y) = \operatorname{AND}(a, b). $$

The claim is that if Alice and Bob follow a deterministic strategy when they each produce their bit, the game can only be won at most 75% of the time. If Alice and Bob use a stochastic strategy to produce their bit, they can also only win at most 75% of the time. Here is some code to check this. First, you import Numpy and set the seed of the random number generator.

import numpy as np
np.random.seed(0)

Then you introduce a couple of constants:

N = 64000
BOOLEANS = [False, True]

Python does not have a built-in exclusive-or function or operator, so here is my version of the exclusive-or function:

def xor(a, b):
    return np.not_equal(a, b)

The following function determines whether the game is won:

def is_game_won(a, b, x, y):
    return np.equal(xor(x, y), np.bitwise_and(a, b))

There are four deterministic strategies to produce a single bit from a given bit:

def strategy_0(x):
    return np.full((x.size,), False)

def strategy_1(x):
    return np.full((x.size,), True)

def strategy_2(x):
    return x

def strategy_3(x):
    return np.bitwise_not(x)

Here strategy_0 and strategy_1 are the constant boolean functions, strategy_2 is the identity function, and strategy_3 is the not function. There is also the stochastic strategy, which we are going to assume is unbiased and thus uses a uniform distribution:

def strategy_4(x):
    return np.random.choice(BOOLEANS, (x.size,))

To calculate the fraction of games won, you can use the following function:

def fraction_of_wins(f, g, N: int) -> float:
    a = np.random.choice(BOOLEANS, (N,))
    b = np.random.choice(BOOLEANS, (N,))
    games = is_game_won(a, b, f(a), g(b))
    wins = games[games]
    return wins.size / N

This function takes two functions f and g that correspond to the strategies employed by Alice or Bob in producing their own bits. Finally, you check all possible combinations of strategies:

strategies = [strategy_0, strategy_1, strategy_2, strategy_3, strategy_4]

f = np.empty((5,5), dtype=float)

for i in range(5):
    for j in range(5):
        f[i,j] = fraction_of_wins(strategies[i], strategies[j], N)

print(f)

The output is:

[[ 0.75101562  0.25125     0.75078125  0.250375    0.49703125]
 [ 0.245375    0.75110937  0.25189062  0.74864062  0.50015625]
 [ 0.7499375   0.25082813  0.24928125  0.74746875  0.49771875]
 [ 0.24970312  0.75384375  0.75096875  0.2478125   0.49728125]
 [ 0.50148437  0.500125    0.5033125   0.50159375  0.50101562]]

The largest fractions are close to 75%. When either Alice or Bob (or both) use the stochastic strategy, the fraction of wins is close to 50%. In more detail, Alice and Bob have the largest chance of winning if they follow one of the following strategies:

  • Both always return 0
  • Both always return 1
  • Alice (Bob) always returns 0 and Bob (Alice) returns the same as given
  • Alice (Bob) always returns 1 and Bob (Alice) returns the negation of what was given
  • Alice (Bob) returns the same as given and Bob (Alice) returns the negation of what was given

The other strategies result in a smaller winning probability.
